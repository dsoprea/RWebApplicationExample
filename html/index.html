One basic way to quickly see how a vector is distributed is to plot a histogram. This sorts your dataset into a set of buckets that each represent a slice of the range. For this example, you'll generate a dataset and plot a histograph.
Example: 
x <- rnorm(100)
h <- hist(x)
Try it yourself:
Result:
A density plot takes a histogram made to reflect the portion of samples found in each bucket and reduces it to a curve. For this example, we'll plot a histogram and draw a density-curve on top of it.
Example: 
x <- rnorm(100)
h <- hist(x, probability=T)
d <- density(x)
lines(d)

A simple linear model will calculate a linear equation that best fits your data (using a single independent variable/predictor/regressor). For this example, we'll use the builtin cars dataset.
Example: 
plot(speed ~ dist, data=cars)
cars.lm = lm(speed ~ dist, data=cars)
abline(cars.lm)

A spline is a set of separate curves calculated to produce a better fit to the data. For this example, we'll plot a spline with degrees-of-freedome (4) through a data-frame in the DAAG dataset.
Example: 
library(DAAG)
library(splines)

plot(thickness ~ distance, data=geophones)

df <- 4
lmb <- lm(thickness ~ bs(distance, df), data=geophones)
ord <- order(geophones$distance)
lines(geophones$distance[ord], fitted(lmb)[ord])
